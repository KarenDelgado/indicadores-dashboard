# indicadores-dashboard

Aplicación web desarrollada con HTML, CSS y JavaScript para cargar archivos Excel `.xlsx`, seleccionar mes y zona, y visualizar indicadores operativos de forma automática.

## Funciones

- Carga de archivos Excel `.xlsx`
- Lectura de hojas DATOS y Estadistica Real
- Selección de mes
- Selección de zona
- Cálculo automático de indicadores
- Visualización en tarjetas
- Animación de carga
- Funcionamiento local en navegador
